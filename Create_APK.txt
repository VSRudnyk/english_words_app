1. Установите EAS CLI:
   Если EAS CLI ещё не установлен, выполните команду:
   npm install -g eas-cli

2. Войдите в свою учётную запись Expo:
   Убедитесь, что вы вошли в свою учётную запись Expo:
   eas login

3. Инициализируйте EAS Build:
   Если вы ещё не инициализировали EAS в проекте, выполните:
   eas build:configure
   Следуйте инструкциям, чтобы настроить проект. Выберите "Managed Workflow", если вы используете Expo без отъединения.

4. Создайте APK:
   Для создания APK выполните:
   eas build --platform android --profile preview
   После завершения сборки вы получите ссылку для скачивания APK.

5. Тестируйте APK:
   Скачайте APK и установите его на своё устройство для тестирования.

6. Для создания AAB (Android App Bundle) для публикации в Google Play:
   Используйте команду:
   eas build --platform android --profile production
   Убедитесь, что у вас настроен ключ подписи для публикации.
